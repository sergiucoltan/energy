My projects

ADLV Music Academy

Ad Libitum Voices Music Academy

2021

  • ADLV Music Academy is a demo web app for a music school, created using ASP.NET MVC.
  • The approach was database first, I have used a Repository class, Microsoft SQL Server for the database, Bootstrap and jQuery for design.
  • The purpose of this app is the registration, subscription of students to music courses and also viewing courses,after creating an account. The app is also built to be used by teachers, with a different authorization, to create lessons and modify courses and also by an admin or a manager who has all the permissions to create, modify and erase data.
